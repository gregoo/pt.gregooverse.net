Find items

  • OS : Windows
  • Language : English
  • Programming language : C#
  • Targetted .NET Framework : 2.0

This tool reads every character .dat file and every warehouse .war files inside your dataserver.
It extracts all the item names and header/checksum couples from the inventory and the warehouse that match the searched term.
Once it has retrieved all the values, it generates a log of all the items.

Example

Phoenix Armor              58527714                2596476058              Character2
Scale Armor             49496927                635998295               Character3
Synthetic Armor         64323725                2799058463              Character, Copy of Character
Frenzy Armor            6136300         2701650548              Character, Copy of Character
Phoenix Armor           25328568                2638931165              Character, Copy of Character
