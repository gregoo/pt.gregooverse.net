Dupe checker

  • OS : Windows
  • Language : English
  • Programming language : C#
  • Targetted .NET Framework : 2.0

This tool reads every character .dat file and every warehouse .war files inside your dataserver.
It extracts all the item names and header/checksum couples from the inventory and the warehouse.
Once it has retrieved all the values, it generates a log of all the duplicated items.
